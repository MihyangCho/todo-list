# Matrix · Todo

アイゼンハワーマトリクスで、タスクを「緊急度」と「重要度」で整理するシンプルなToDoアプリです。


---

## 概要

> 「重要なことは緊急ではなく、緊急なことは重要ではない」— ドワイト・アイゼンハワー

タスクを追加して「緊急」「重要」フラグを設定するだけで、自動的に4つの象限に振り分けられます。何を今やるべきか、何を後回しにできるかを一目で把握できます。

---

## 機能

- **リストビュー** — タスクの追加・完了・削除。緊急／重要フラグでフィルタリング可能
- **マトリクスビュー** — アイゼンハワーマトリクス（2×2）でタスクを可視化
- **ドラッグ＆ドロップ** — 各象限内でタスクの優先順位を並び替え
- **自動保存** — `localStorage` によりページを閉じてもデータが保持される

---

## 4つの象限

| 象限 | 緊急 | 重要 | 対応方針 |
|------|------|------|----------|
| Ⅰ 今すぐ実行 | ✅ | ✅ | 自分で今すぐ対処する |
| Ⅱ 計画を立てる | ❌ | ✅ | スケジュールを決めて取り組む |
| Ⅲ 委任する | ✅ | ❌ | 他の人に任せる |
| Ⅳ 意識的に管理 | ❌ | ❌ | 時間を制限し徐々に減らす |

---

## 技術スタック

- **React 18** (CDN / Babel Standalone)
- **localStorage** によるデータ永続化
- **HTML5 Drag & Drop API**
- ビルドツール不要 — 単一の `index.html` で動作
